# Noce Mobili — Instruções para Claude Code

## Git
- Sempre commitar e fazer push **diretamente no branch `main`**
- Não criar branches separadas
- Não criar pull requests — push direto para `main`

## Projeto
- Express.js + Vanilla JS, sem build step
- Todo conteúdo editável via `/admin` (autenticado)
- Dados em `/data/content.json`
- Imagens processadas pelo Sharp e salvas em `/public/images/`
